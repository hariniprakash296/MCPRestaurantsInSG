# MCPRestaurantsInSG
MCP to find restaurants selling a specific food in SG
1. Accepts a **food/restaurant query**.
3. Uses the **Google Places API** to search for places in **Singapore**.
4. Returns **cleaned and structured results**, including:
   
   a. Name  
   b. Address  
   c. Rating  
   d. Price level
